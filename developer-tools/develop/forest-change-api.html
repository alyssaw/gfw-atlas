http://api.globalforestwatch.org/forest-change

Users can make calls to the GFW API and receive statistics on:

  • University of Maryland (UMD)/Google tree cover loss & gain
  • FORMA alerts
  • IMAZON SAD alerts
  • QUICC alerts
  • Terra-i alerts
  • NASA active fires

They can post their own geometry, or use predefined geometries such as administrative boundaries, concessions or protected areas.

They can also specify the time period for they are interested in.

Full documentation of the calls are here

https://github.com/gfw-api
